# Safety Protocol

Use this reference when triaging symptoms, medication reactions, or abnormal test results.

## Escalation Bands

### Emergency now

Use this when the user reports possible emergency symptoms, including:

- chest pain with pressure, fainting, or shortness of breath
- one-sided weakness, facial droop, new confusion, trouble speaking
- severe trouble breathing
- lips or tongue swelling
- severe allergic reaction
- uncontrolled bleeding
- seizure
- suicidal intent or imminent self-harm risk

Response pattern:

- lead with emergency instruction
- keep it short
- avoid extended analysis

### Urgent same day

Use this when the case may need prompt clinician review but does not clearly sound like an emergency:

- high fever with significant weakness
- dehydration concern
- worsening infection symptoms
- new medication reaction without airway symptoms
- persistent severe pain
- significant abnormal home vitals if provided

### Routine soon

Use this for issues that need clinician follow-up but not immediate escalation:

- ongoing but stable symptoms
- mild abnormal labs without red flags
- medication tolerance questions
- chronic condition monitoring questions

### Education only

Use this when the user is mainly asking for explanation, definitions, or appointment prep and there are no warning signs in the provided information.

## Triage Discipline

- Triage the care setting, not the diagnosis.
- Prefer a conservative escalation threshold when the history is incomplete.
- If details are sparse, ask for the minimum facts that change urgency:
  - age group
  - symptom onset
  - severity
  - relevant conditions
  - pregnancy or immunocompromise when relevant
- Do not bury red flags below long explanations.
- If the user names a diagnosis already given by a clinician, you may explain it in plain language, but do not confirm or revise it yourself.
